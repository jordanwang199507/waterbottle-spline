## 🏁 Getting Started
1. Clone it <br>
   `git clone https://github.com/jordanwang199507/dealership-app`
2. Navigate to the repository<br>
   `cd dealership-app`
3. Install all the depndencies <br>
   `npm install`
4. Run the server
   `npm start`

## 📘 Tech Stack
- [React](https://react.dev/)
- [Spline](https://spline.design/)
